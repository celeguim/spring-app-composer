# spring-app-composer
Java stack (Spring, MySQL, MemCache, RabbitMQ)

# Prerequisites
- JDK 11 
- Maven 3 
- MySQL 8

# Technologies 
- Spring MVC
- Spring Security
- Spring Data JPA
- Maven
- JSP
- Tomcat
- MySQL
- Memcached
- Rabbitmq
- ElasticSearch

# Database
Mysql DB 
Database initialization:
- /src/main/resources/db_backup.sql
- > mysql -u <user_name> -p accounts < db_backup.sql
